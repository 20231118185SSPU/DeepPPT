### 下载 PPTX

每个项目的 `exports/` 目录包含可直接打开的 PPTX 文件：

```

```

## Project Structure

```
<project>/
├── design_spec.md       # Design specification (human-readable)
├── spec_lock.md         # Machine-readable execution contract
├── images/              # Image assets (AI-generated + web-sourced)
│   ├── *.png            # AI-generated images
│   └── web_assets/      # Web-sourced images for deep-dive pages
├── notes/               # Speaker notes per page
├── svg_output/          # Original SVGs (with placeholders)
└── exports/             # PPTX output (DeepPPT examples only)
```

## Preview SVG locally

```bash
python -m http.server --directory examples/<project>/svg_output 8000
# visit http://localhost:8000
```
